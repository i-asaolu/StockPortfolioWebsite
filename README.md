MAKE SURE YOU HAVE PYTHON INSTALLED!

1. Open your terminal and navigate to the directory you plan on working from.
2. Clone the repo "git clone https://github.com/Iasaolu/Stock-Portfolio-Website.git"
3. cd stocks
4. python3 -m venv venv
5. pip install django
6. pip install requests
7. source venv/bin/activate
8. pip install -r requirements.txt
9. python manage.py migrate (if needed)
10. python manage.py runserver
11. Open browser and run http://127.0.0.1:8000/
